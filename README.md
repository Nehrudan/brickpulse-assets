# brickpulse-assets
BrickPulse – Official Asset Repository
BrickPulse ist ein futuristischer Energiekristall-Token, entworfen als spielerischer Wertträger für gemeinsame Projekte, Minigames und Community-Erlebnisse.
Das Icon zeigt einen volumetrischen Kristall mit einem vertikalen Neonpuls im Inneren – als Symbol für die Energie und Dynamik des Projekts.

Dieses Repository enthält:
• das offizielle Token-Icon (PNG/SVG)
• Branding-Elemente für Wallets und Plattformen
• optionale Metadaten für Token-Tools und Solana-Integrationen
